
# Java Dragon Quest 1 Remake

This project is a faithful remake of Dragon Quest 1 (Dragon Warrior 1) in Java 8, using only standard libraries (Java 2D, AWT, Swing, and Java Sound API). No external dependencies are required.

## Features

- Classic turn-based RPG gameplay inspired by the original NES title
- Pixel-art graphics and authentic sound effects
- Multiple endings and story paths
- Save/load functionality
- Cross-platform runnable JAR and Windows executable

## Getting Started

**Download:** [Release v0.0.3](https://github.com/leonardo-ono/JavaDragonQuest1/releases/tag/v0.0.3)

**Requirements:** Java 8 or newer

**How to Run:**
- Double-click the JAR file or run from terminal: `java -jar JavaDragonQuest1.jar`
- For Windows, use the provided executable

## Controls

- Arrow keys: Move player
- X: Confirm/Action
- Z: Cancel/Back
